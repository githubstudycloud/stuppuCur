# Vue Enterprise Base 项目总结

## 🎯 项目概述

Vue Enterprise Base 是一个专为大厂多项目管理设计的现代化Vue项目基座，采用最新的技术栈和最佳实践，为企业级应用开发提供完整的解决方案。

## 🏗️ 技术架构

### 核心技术栈
- **前端框架**: Vue 3.4+ (Composition API)
- **开发语言**: TypeScript 5.3+
- **构建工具**: Vite 5.0+
- **UI组件库**: Element Plus 2.4+
- **状态管理**: Pinia 2.1+
- **路由管理**: Vue Router 4.2+
- **HTTP客户端**: Axios 1.6+

### 开发工具链
- **代码规范**: ESLint + Prettier
- **Git钩子**: Husky + lint-staged
- **测试框架**: Vitest
- **类型检查**: Vue TSC
- **自动导入**: unplugin-auto-import
- **组件注册**: unplugin-vue-components

```env

```

- **端口**: 3000
- **热重载**: 启用
- **API代理**: 配置到后端服务
- **HTTPS**: 支持本地HTTPS

- **输出目录**: dist/
- **资源压缩**: 启用
- **源码映射**: 生产环境禁用
- **文件命名**: 带hash的文件名

## 🚀 部署方案

### 1. 传统部署
```bash
npm run build
# 将dist目录部署到Web服务器
```

### 2. Docker部署
```bash
docker build -t vue-enterprise-base .
docker run -p 80:80 vue-enterprise-base
```

### 3. Docker Compose
```bash
docker-compose up -d
```

### 4. 自动化部署
```bash
./scripts/deploy.sh production
```
